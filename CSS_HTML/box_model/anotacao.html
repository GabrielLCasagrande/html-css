Aninhamento se refere a uma caixa dentro de outra caixa


A altura de uma caixa é Height
A largura é Width
Podemos traçar uma linha em torno da caixa usando Border
Podemos criar um espaçamento entre a borda e a palavra (interno), chamado Padding
Podemos criar um espaço entre caixas, caso tenha um elemento embaixo, criando um espaço externo chamado Margin
Podemos criar um contorno também,fora do elemento chamado Outline

Tipos de Caixa (box-level) (inline-level)

Box-level: Paragráfo aqui que continua na linha de baixo dessa forma o box level [cria uma caixa embaixo do paragráfo, numa linha nova, ocupa a largura total da tela]
ondea próxima linha fica embaixo

Inline-level: Paragráfo aqui que continua na linha de baixo dessa forma o inline [cria uma caixa que não quebra a linha] e não ocupa a largura total, podendo escrever o texto na frente dele

box-level tag:div, h1-h6, p, main, header, nav, article, aside, footer, form, video

inline-level:span, a, code, small, strong, em, sup-sub, label, button, input, select
